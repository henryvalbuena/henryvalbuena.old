Data Communication Client-Server API

Designed and developed a client-server application which communicates to a remote server using sockets and TCP protocol. The idea behind it is to experiment with socket programming and how two terminals can exchange data effectively. The client application was built with a combination of C and C++. C is used to access the platform libraries that contains all the necessary methods to establish data communication using TCP and sockets. C++ is used for the rest of the modules that compose the client application, there is a wrapper which has methods to create the socket file descriptor, connect, send, receive data and close the connection. Additionally, a special module which is used to transform strings into JSON format since the data transfer between the terminals is standardized by the using JSON to exchange data effectively across platforms. The server-side program is done using node.js and built in methods to create sockets to listen for incoming connections, receive and send data.

Maze Solving Application

Developed an application which takes an image of squared shaped size L * L with black and white colors that represent the walls (black) and the path (white) for the maze. The program checks the maze and using three / graph algorithms like Deep First Search find the way out from the starting position. Making use of Java and the standard libraries the image is read from the file system, read pixel by pixel to determine the walls and the path and then using Lists and graph algorithms construct the available path to return a copy of the image with the solution marked down in color.
